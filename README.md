# Vite / Vue 3 Template:

Create your website with this ready to use template using Vite.js and Vue.js

Dependencies used :

```vue: 3.2.45```

```vue-router: 4.1.6```

```sass: 1.58.3```

```vite: 4.1.0```

```pinia: 2.0.33```

# Quick Install :

1. Click on ``` Use this template ```

2. Clone your own template

3. Go inside your project 

4. Use ```npm install``` to install all dependencies

5. Use ```npm run dev ```
